Maintaining Linked Data via simple Web forms


The RaUL framework let users interact with RDF data over simple Web forms. See it in action here. The Web service can automatically render Web forms with a binding to an RDF graph maintained via RDFa annotations. The data in the Web form is automatically mapped to concepts in a Web ontology. By the virtue of typing Web form elements via RDF ontologies, existing Linked Data can be fetched at run-time from any Web resource to pre-populate a Web form. more

RaUL

RaUL, the RDFa User Interface Language, is a user interface markup ontology that can be used to describe the structure of a Web form as RDF statements. Learn about the semantic concepts behind RaUL.

ActiveRaUL

A generic Web service, that coupled with a generic client servlet, generates a user interface model for semantic Web applications. Learn about the ActiveRaUL Web service functionality and its endpoints.
